.. _PRINT_BB_STATS:

PRINT_BB_STATS
-------------

Enabling this parameter will result in printing of statistics on the number of
instructions, memory instructions and basic blocks in each loop or function.

Category
+++++++++

Miscellaneous

Value Type
+++++++++++

Integer

Valid Values
+++++++++++++

0, 1

Default Value
++++++++++++++

unset (0)

Location Where Default is Specified
+++++++++++++++++++++++++++++++++++

``examples/legup.tcl``

Dependencies
+++++++++++++

None

Applicable Flows
+++++++++++++++++

All devices and flows

Test Status
++++++++++++

Actively in-use

Examples
+++++++++

    ``set_parameter PRINT_BB_STATS 1``
