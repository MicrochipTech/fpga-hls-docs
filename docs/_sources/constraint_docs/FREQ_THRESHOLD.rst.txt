.. _FREQ_THRESHOLD:

FREQ_THRESHOLD
--------------

Minimum pattern frequency written to dot/v file

Category
+++++++++

Miscellaneous

Value Type
+++++++++++

Integer

Valid Values
+++++++++++++

Positive integer

Default Value
++++++++++++++

unset (0)

Location Where Default is Specified
+++++++++++++++++++++++++++++++++++

``examples/legup.tcl``

Dependencies
+++++++++++++

At least one of these parameters: :ref:`PS_WRITE_TO_DOT` or :ref:`PS_WRITE_TO_VERILOG` needs to be enabled.

Applicable Flows
+++++++++++++++++

All devices and flows

Test Status
++++++++++++

Actively in-use

Examples
+++++++++

    ``set_parameter FREQ_THRESHOLD 1``
