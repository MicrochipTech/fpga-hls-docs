set_smarthls_config_board
-------------

The name of the board being targetted by SmartHLS. This constraint is set in the 
set_project function and should not be set independently.

Category
+++++++++

Board and Device Specification

Value Type
+++++++++++

String

Valid Values
+++++++++++++

Any valid directory name

Default Value
++++++++++++++

DE1-SoC

Location Where Default is Specified
+++++++++++++++++++++++++++++++++++

``examples/smarthls.tcl``

Dependencies
+++++++++++++

None

Applicable Flows
+++++++++++++++++

All devices and flows

Test Status
++++++++++++

Actively in-use

Examples
+++++++++

    ``set_smarthls_config_board DE1-SoC``
