.. _set_memory_local:

set_memory_local
-----------------

This parameter can be used to force a particular array to be placed in local memory instead
of global memory.

Category
+++++++++

HLS Constraints

Value Type
+++++++++++

String

Valid Values
+++++++++++++

Array name

Default Value
++++++++++++++

None

Location Where Default is Specified
+++++++++++++++++++++++++++++++++++

``examples/legup.tcl``

Dependencies
+++++++++++++

LOCAL_RAMS

Applicable Flows
+++++++++++++++++

All devices and flows

Test Status
++++++++++++

No longer in-use

Examples
+++++++++

``set_memory_local JpegFileBuf``
