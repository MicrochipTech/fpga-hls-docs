.. _PRINT_STATES:

PRINT_STATES
-------------

When this parameter is enabled, the cur_state of each function will be
displayed at every clock cycle in simulation.

Category
+++++++++

Simulation

Value Type
+++++++++++

Integer

Valid Values
+++++++++++++

0, 1

Default Value
++++++++++++++

unset (0)

Location Where Default is Specified
+++++++++++++++++++++++++++++++++++

``examples/legup.tcl``

Dependencies
+++++++++++++

None

Applicable Flows
+++++++++++++++++

All devices and flows

Test Status
++++++++++++

Actively in-use

Examples
+++++++++

    ``set_parameter PRINT_STATES 1``
